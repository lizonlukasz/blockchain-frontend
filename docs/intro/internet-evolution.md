# Rozwój internetów


### Web 1.0
- od ok 1994 
- Internet tylko do odczytu
- Brak interakcji użytkownika


### Web 2.0
- od ok 2004
- Interakcje a użytkownikiem
- Rozwój technologii sieciowych - JS, HTML, CSS

### Web 3.0
- technologia AI
- blockchain i aplikacje o niego oparte
- Decentralizacja sieci
- IoT
